# Image directory

Drop all brand and editorial assets in this folder. The site is wired to these exact paths.

## Brand assets

- `logo.png` — primary HeyPearl wordmark, transparent PNG
- `logo-inverse.png` — light-on-dark version (used in footer)
- `pearl-mark.svg` — the layered P.E.A.R.L. system mark (NOT a literal pearl)
- `misti-headshot.jpg` — founder portrait used on the homepage and about page

## Open Graph

- `og-default.jpg` — default social share image, 1200×630

## Favicons (place in `public/`, not here)

- `favicon.ico`
- `apple-touch-icon.png` (180×180)
- `icon-192.png`, `icon-512.png` for PWA

See `IMAGE_PROMPTS.md` in the project root for cinematic prompts and full asset list.
